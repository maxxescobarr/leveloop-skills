# 📚 Leveloop Skills — Lienzos de Fe

Skills del agente de IA de Lienzos de Fe. Cada carpeta contiene un `SKILL.md` con instrucciones completas para el agente.

## Skills disponibles

| Skill | Descripción | Responsable |
|---|---|---|
| `crear-pedido/` | Crear pedidos, liquidaciones y actualizar direcciones en Shopify + Notion + ManyChat | Fer, Max |
| `shopify-consultas/` | Consultar ventas, órdenes y datos de Shopify | Max |
| `atencion-cliente/` | Guías de respuesta y flujos de atención al cliente | Fer |

## Cómo funciona

El agente lee el `SKILL.md` correspondiente antes de ejecutar cualquier tarea.
Las actualizaciones las hace el agente automáticamente al final de cada mejora confirmada.
